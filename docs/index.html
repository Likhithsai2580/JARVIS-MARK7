JARVIS MARK 7 Documentation

Welcome to the documentation for JARVIS MARK 7. This project is an advanced AI control system that combines face authentication, voice commands, and automated task execution.

Client Documentation

The client is a React-based application that provides a modern GUI for interacting with the JARVIS MARK 7 system. It uses Tailwind CSS for styling and WebSocket for real-time communication.

Setup

  1. Clone the client repository: git clone https://github.com/Likhithsai2580/JARVIS-MARK7-CLIENT.git
  2. Navigate to the client directory: cd JARVIS-MARK7-CLIENT
  3. Install dependencies: npm install
  4. Start the development server: npm run dev

Project Structure

    .
    ├── src/               # Source code
    └── public/            # Static assets

Server Documentation

The server is a Python-based application that handles code execution, face authentication, and communication with the Android bridge server. It uses FastAPI for the API and WebSocket for real-time communication.

Setup

  1. Clone the server repository: git clone https://github.com/Likhithsai2580/JARVIS-MARK7-SERVER.git
  2. Navigate to the server directory: cd JARVIS-MARK7-SERVER
  3. Set up the Python environment:
    1. Create and activate virtual environment: python -m venv venv
    2. Activate the virtual environment: source venv/bin/activate (Linux/Mac) or .\venv\Scripts\activate (Windows)
    3. Install dependencies: pip install -r codebrew/requirements.txt
  4. Set up the Android Bridge Server:
    1. Navigate to the android_bridge_server directory: cd android_bridge_server
    2. Install dependencies: npm install
  5. Configure environment variables:
    1. Copy example env files: cp .env.example .env
    2. Copy example env files for android bridge server: cp android_bridge_server/.env.example android_bridge_server/.env

Project Structure

    .
    ├── codebrew/         # Python code execution engine
    ├── android_bridge/   # Android bridge server
    └── face_auth/        # Face authentication service
